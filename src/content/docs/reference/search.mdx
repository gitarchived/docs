You can search repositories by name and owner

```http
GET https://api.gitarchived.org/search?q=query
```

### Response
```json
{
  "results": [{ repository struct }],
  "message": "OK",
  "status": 200
}
```

### Query parameters
| Parameter | Description | Example |
| :--- | :--- | :---
| q | Search query | svelte
