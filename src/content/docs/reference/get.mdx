---
title: Get repository
description: Get a repository
---

You can add a repository with a `GET` request to `/<host>/<owner>/<repo>`.

```http
GET https://api.gitarchived.org/github/gitarchived/api
```

### Response
```json
{
  "data": { repository struct },
  "message": "OK",
  "status": 200
}
```

### Query parameters
| Parameter | Description | Example |
| :--- | :--- | :---
| host | The host of the repository | github |
| owner | The owner of the repository |  gitarchived |
| repo | The name of the repository | api |

*You can use only [supported hosts](/information/hosts) url*
